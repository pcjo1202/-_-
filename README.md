## 기능

1.  <code>set</code>

    - 처음 기본 값을 설정하는 함수

2.  <code>clear</code>

    - value 값을 0으로 리셋

3.  <code>add</code>

    - <code>set()</code>를 통해 설정한 value 값에 값을 더하는 함수

4.  <code>subtract</code>

    - <code>set()</code>를 통해 설정한 value 값에 값을 빼는 함수

5.  <code>multiply</code>

    - <code>set()</code>를 통해 설정한 value 값에 값을 빼는 함수

6.  <code>divide</code>

    - <code>set()</code>를 통해 설정한 value 값에 값을 나누는 함수
